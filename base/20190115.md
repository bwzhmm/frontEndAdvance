1. （必考） 你是如何理解 HTML 语义化的？
      通俗理解：单纯的不带任何样式得HTML代码，来标记标题、代码、强调内容等等；即从代码上就能看出页面结构。
      常用的：
         <h1>~<h6>:标题使用，重要性递减
         <P>段落标记：不用使用<br/>
         <ul>:无序列表  导航条,  <ol>,<li>
         <dl>,<dt>,<dd>,   <dl>:定义列表 <dt>:项目名称 <dd>: 项目描述
         <em>：用作强调， <strong>：重点强调
         <table>,<thead>,<tbody>,<td>,<th>,<caption> :用来做表格 不用来布局
2. meta viewport 是做什么用的，怎么写？
    默认[body]的宽度取自[HTML],而[HTML]宽度取自[viewport],[viewport]的宽度正好等于[浏览器]窗口宽度
    document.documentElement.clientWidth取的是[viewport]的宽。而window.innerWidth能取到[浏览器]窗口的宽
    pc端两者区别 window.innerWidth包含滚动条的宽。
    移动端：window.innerWidth来取得visual viewport的宽。  document.documentElement.clientWidth获取layout viewport的宽度尺寸，总是固定的。


    viewport meta标签：
    width：控制 layout viewport 的大小。
    height 和 width相对应，指定高度。
    initial-scale:初始缩放比例，也即是当页面第一次load的时候缩放比例
    maximum-scale:允许用户缩放到的最大比例.
    minimum-scale:允许用户缩放到的最小比例。
    user-scalable:用户是否可以手动缩放。

    width  = device-width :浏览器将当前页面的layout viewport宽度设置为设备的ideal viewport ；
    inital-scale： 页面初始缩放设置一个有意义的值，相对于ideal viewport，产生了visual viewport宽度
    ，再计算layout viewport的宽度值。

    flexible.js：
    1，根据设备定义dpr  ，iPhone 1,2,3 倍 方案
    2，利用rem设置页面元素大小  
    比如设计稿 750px，默认分成10份，75px像素对应1个rem。
